# mp4_parser
Pull information from MP4 metadata.  Data isn't returned yet, nor is it in a usable configuration.'
Only translates:
* ftyp
* moov
* mdat
* mvhd
* trak
* tkhd
* mdia
* mdhd
* hdlr
* minf
* vmhd
* smhd
* dinf
* stbl
* dref
* url 
* stts
* ctts
* stss
* stsz
* stsc
* stco
* sdtp
* stsd
* free
* edts
* udta

The following types are recognized but not translated in any way:
* meta
* elst
* wide
* iods
* SDLN
